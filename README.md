# Cyber Garden School 2025

## Наш кейс

От: **ООО НМФ «Нейротех»**

Название: **Система дополненного управления при помощи биомеханических сигналов «CallibriControl»**

### Описание кейса

#### Идея проекта
CallibriControl — это интеллектуальная система распознавания движений,
основанная на данных электромиографии (ЭМГ) и инерциальных датчиков (МЭМС/IMU) многофункционального датчика Callibri.
Приложение анализирует паттерны мышечной активности,
ускорения движений и положения датчика в пространстве в реальном времени и преобразует их в команды и сочетания клавиш клавиатуры.
Это позволяет пользователям управлять компьютером и приложениями естественными движениями и жестами,
обеспечивая доступное управление для людей с ограниченными возможностями,
а также создавая новые парадигмы взаимодействия между человеком и компьютером.

#### Общее описание проекта

Беспроводной датчик «Callibri» (https://callibri.com/ru) осуществляет регистрацию
функциональной активности мышц пользователей (ЭМГ-сигнал) и положения датчика в пространстве (МЭМС).
Зарегистрированные данные с целью дальнейшей обработки передаются посредством протокола Bluetooth LE в программу на ПК.
Собственный SDK (https://sdk.callibri.com/) позволяет легко создавать приложения,
предоставляя разработчику инструмент для обработки в своем коде первичных данных мышечной и двигательной активности.
Это позволяет получать объективные данные, позволяя участникам оценивать физическую активность.

#### Цель разработки

Создать приложение, которое поможет пользователям:

- Управлять компьютером и приложениями с помощью движений и мышечных сокращений;

- Привязывать произвольные движения к нажатиям клавиш и их комбинациям;

- Обеспечить альтернативный способ взаимодействия для людей с ограниченной
подвижностью или инвалидностью;

- Создать инновационный интерфейс управления для гейминга и специализированных
приложений;

- Повысить эффективность и удобство управления в определённых профессиональных
сценариях.

Целевая аудитория: люди с ограниченными возможностями опорно-двигательного аппарата,
киберспортсмены, профессионалы, требующие специальных интерфейсов управления,
исследователи в области HCI (Human-Computer Interaction),
пользователи, ищущие инновационные способы взаимодействия.
